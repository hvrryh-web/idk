## What is the SRD Alpha?

The SRD Alpha is a **complete, playable rules reference** for the WUXUXIANXIA tactical TTRPG system. It synthesizes design documents, chat logs, and implementation notes into game-ready documentation.

**Key Features**:
- ✅ Complete character creation rules
- ✅ Full stat system (9 Primary + 3 Core + 3 Aether)
- ✅ SCL (Soul Core Level) progression system
- ✅ Combat resources (THP, AE, Strain, Guard)
- ✅ Three conflict pillars (Violence, Influence, Revelation)
- ✅ 4-Step condition ladders
- ✅ Seven Quick Actions
- ⏳ Technique system (coming in Patch 0.3)
- ⏳ Equipment and artifacts (coming in Patch 0.3)
- ⏳ Full GM guidelines (coming in Patches 0.4-0.5)

## How to Use These Documents

### For New Players

**Read in this order**:
1. **Patch 0.1, Section 0** - Core concepts (15 min)
2. **Patch 0.1, Section 1** - Character creation, pick a Playbook (20 min)
3. **Patch 0.2, Section 3** - Combat resources (15 min)
4. **Patch 0.2, Section 4** - Conflict types (15 min)

**Total**: ~65 minutes to basic competence

**Then**:
- Create your first character using a Playbook
- Run a simple combat with 1-Beat mode
- Graduate to 3-Stage Combat after 2-3 sessions

### For Experienced TTRPGers

**Skim in this order**:
1. **SRD History** - See design evolution (skim 10 min)
2. **Patch 0.1** - Stat system and SCL (skim 20 min)
3. **Patch 0.2** - Combat resources and pillars (skim 15 min)

**Total**: ~45 minutes to system fluency

**Then**:
- Build custom character (ignore playbooks)
- Use 3-Stage Combat from start
- Experiment with multi-pillar builds

### For Game Masters

**Read everything in order**:
1. **SRD History** - Understanding design intent (30 min)
2. **SRD Structure** - See complete scope (45 min)
3. **Patch 0.1** - Character creation rules (60 min)
4. **Patch 0.2** - Combat and conflict mechanics (50 min)

**Total**: ~3 hours to full system knowledge

**Then**:
- Reference Section 2.6 (Balance Caps) when reviewing PCs
- Reference Section 3 (Resources) during combat prep
- Reference Section 4 (Conflict Types) for encounter variety

### For Designers/Homebrewers

**Start with**:
1. **SRD History** - See how system evolved (30 min)
2. **SRD Structure** - Understand framework (45 min)
3. Skim patches for specific mechanics

**Then**:
- Use balance caps (2 × SCL) for custom techniques
- Follow Power Profile system for custom archetypes
- Maintain "Power Draws Blood" cost track philosophy

## Feedback and Playtesting

### Reporting Issues

**Found errors?**
- Check if it's noted in "Playtest Note" sections
- Report to design team or GM
- Note patch number and section

**Found balance issues?**
- Try 3-5 sessions before reporting
- Note specific scenarios (character builds, encounters)
- Include SCL range where issue occurred

**Found unclear rules?**
- Check glossary (Section 12, coming in Patch 0.5)
- Ask GM or fellow players first
- Report persistent confusion points

### Playtesting Focus

**Patch 0.1-0.2 Playtest Goals**:
- Can players create characters in <1 hour?
- Is SCL calculation intuitive?
- Do the 4 combat resources create interesting choices?
- Are Quick Actions useful and clear?
- Do all three conflict pillars feel equally viable?

**What to Track**:
- Character creation time
- Combat encounter length (rounds and real time)
- Resource depletion rates (THP, AE, Strain)
- Condition application frequency
- Player satisfaction with pillar diversity
